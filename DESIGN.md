# Design System

## Visual Theme

BitGates uses a warm academic canvas, protocol-green surfaces, controlled HKU-red accents, wide-tracked identity type, and node-based layouts. The site should feel like a university research atelier meeting a Hong Kong financial infrastructure briefing.

## Color Palette

Use OKLCH tokens only in production CSS.

- Canvas: warm ivory, not pure white.
- Ink: green-tinted near-black, not pure black.
- Protocol Green: deep green for authority and long-termism.
- HKU Red: restrained action and identity accent.
- Mist: pale green-gray for structural panels.
- Brass: muted warm accent for small metadata.

## Typography

Primary English stack: Avenir Next, Helvetica Neue, Arial, system sans.
Primary Chinese stack: PingFang SC, Hiragino Sans GB, STHeiti, Microsoft YaHei, sans-serif.
Serif accent stack for manifesto quotes: Songti SC, STFangsong, Georgia, serif.

Use wide tracking for BitGates identity moments and compact sans for navigation. Body copy must stay below 75ch.

## Layout

Use a strict but expressive 12-column grid. Avoid centered template stacks. Favor asymmetric chapter sections, long horizontal evidence strips, node diagrams, image mosaics, and staggered program rails.

## Components

- Chapter header with section number, Chinese title, English title, and concise bilingual copy.
- Evidence ledger for facts and metrics.
- Program rail with four modules, not identical icon cards.
- Case mosaic with real activity images.
- Research report archive with share, export, and print actions.
- Team portrait panel with bilingual credentials and collapsible perspective text.
- Governance node map.
- Join matrix for students, enterprises, and advisors.

## Motion

Use subtle load reveal, scroll-linked visibility, hover lift via transform, and non-layout opacity or transform transitions. Respect prefers-reduced-motion.
